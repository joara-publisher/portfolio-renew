Details

프로젝트 배경
법무법인 설립과 동시에 브랜드 신뢰도를 높일 수 있는 공식 홈페이지가 필요함
형사, 이혼, 민사, 성범죄, 음주 등 다양한 법률 분야별 전문 사이트 제작 요구
작업 범위
메인 홈페이지 구축 및 오픈 이후 유지보수 전담
분야별 특화된 서브 사이트 제작 및 콘텐츠 구성
기존 사이트에서 가져온 더티 코드(Dirty Code)를 클린 코드로 리팩토링하여 유지보수성 향상
PHP 기반 캘린더를 flatpickr-calendar로 교체하고, 공공 API 연동을 통한 공휴일 자동 표시 기능 구현
성과 / 특징
다양한 법률 분야에 맞춘 서브 사이트 제작으로 브랜드 전문성 강화
코드 리팩토링을 통해 유지보수 효율성과 협업 생산성 향상
캘린더 개선 및 공휴일 자동 표시 기능으로 사용자 편의성(UX) 강화
짧은 기간 내 안정적인 웹 인프라 구축으로 신설 조직의 서비스 기반 마련
