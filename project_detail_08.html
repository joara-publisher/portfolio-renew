Details

프로젝트 배경
회사 창립 시점에 맞추어 브랜드 정체성을 담은 공식 홈페이지 구축 필요
다양한 부서의 요구사항을 반영하면서도 직관적이고 실용적인 웹사이트 제작이 요구됨
작업 범위
기획자, 디자이너와의 협업을 통해 웹사이트 전체 구축 기획 및 구현
기획 의도와 디자인 방향을 충실히 반영하여 개발
datepicker와 공공 API 연동으로 공휴일 자동 반영 기능 구현
구글 3D 지도 삽입 등 사용자 중심 기능 개발
홈페이지 오픈 이후 유지보수 및 기능 개선 전담
성과 / 특징
짧은 기간 안에 다양한 부서의 니즈를 충족하는 웹사이트 구축
공휴일 자동 반영 등 실용적인 사용자 편의 기능 제공
기획·디자인·개발 간 원활한 협업을 통해 프로젝트 완성도와 신뢰도 제고
